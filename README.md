# Sistema Escolar DWF
Proyecto desarrollado para el Desafío 1 en IntelliJ idea Spring boot.

## Descripción
El Sistema Escolar es una aplicación web que permite gestionar información básica de estudiantes.
## Tecnologías utilizadas
Dependencias Spring Data JPA, H2, Database y Spring Web.
